# Find-Truth-BOM-Bible

This is a personal project where I use software to compare gospel truths in the Book of Mormon and the Bible.

## Inspiration

"In the mouth of two or three witnesses shall every word be established"

The Bible and the Book of Mormon serve as two key witnesses of Jesus Christ.

## Purpose

The goal of this project is to show how the Book of Mormon and the Bible testify of gospel truths together. They build the other up and support one another.
